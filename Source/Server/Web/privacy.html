Privacy Policy

1. Each page request to this webserver gets written to a logfile. The webserver records time and date, IP address, the page/image requested, the referring page (i.e. the page containing the link you clicked on) your browser type and your operating system (your browser sends this information automatically). None of this is sold or released to anyone without prior consent or a court order with the exception of unpersonalised statistics (see 3.).
Most of the web pages have 3rd party content. This privacy policy does not apply to these 3rd parties.

2. The game itself does extensive logging. Everything you do gets recorded. This data is used for debugging purposes and to enforce the Terms and Conditions only. No information from these log files is sold or released to anyone without prior consent or a court order with the exception of unpersonalised statistics (see 3.).

3. Any data collected may be used to build statistics, provided these do not contain any references to personal data.
